# Movie-Ticket-Reservation-System

# Overview

This project is a console-based Movie Ticket Reservation System developed in C++. It simulates real-world cinema booking operations, allowing users to browse movies, select seats, book tickets, and handle payments. It also includes administrative and membership functionalities.

# Features:-

Movie listing with show timings,
Seat selection and booking,
Basic ticket cancellation,
Payment handling (cash and card simulation),
Admin panel to add, update, and delete movies,
User membership system including registration, login, and renewal,
Snacks and drinks ordering with bill generation

# Concepts Used:-

Object-Oriented Programming,
File Handling using fstream,
STL (vectors, sorting),
Input validation and menu-driven programming

# Data Storage:-

Movie data stored in text files,
User and membership data stored in files,
File-based data management approach

# Tech Stack:-

C++,
STL,
File Handling
